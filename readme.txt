This plugin allows guests to edit their comments on your site.

== Description ==

This plugin adds a "Edit" button to your published comments on the site. And allows unregistered users of the site to edit and delete their comments.

You can prohibit the deletion of individual comments and change the appearance of the Edit button.


== Installation ==

1. Copy the plugin folder <strong>mi13-comment-user-edit</strong> to <strong>/wp-content/plugins/</strong>. 
2. Activate the plugin via the <strong> menuPlugins</strong>.
3. That's it.
